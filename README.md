## プライバシーポリシー (Privacy Policy)

本アプリ（JumpBird）におけるユーザー情報の取り扱いについて、以下の通り規定します。

### 1. 個人情報の収集・取得
本アプリは、ユーザーの個人情報（氏名、メールアドレス、デバイス識別子、位置情報など）を一切収集・取得いたしません。

### 2. 外部へのデータ送信
本アプリは、ユーザーのデータを外部サーバー等へ送信することはありません。オフラインでも安心してご利用いただけます。
